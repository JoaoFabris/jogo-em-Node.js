# 🎲 Aventura dos Dados Mágicos

Um jogo de RPG baseado em texto desenvolvido em Node.js, onde você embarca em uma aventura épica através de florestas misteriosas, vilarejos antigos e arenas de combate!

![Node.js](https://img.shields.io/badge/Node.js-43853D?style=for-the-badge&logo=node.js&logoColor=white)
![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=for-the-badge&logo=javascript&logoColor=black)

## 🌟 Sobre o Jogo

**Aventura dos Dados Mágicos** é um jogo de RPG narrativo onde suas escolhas determinam o destino do seu herói. Explore diferentes cenários, lute contra criaturas místicas, colete tesouros e teste sua sorte com o poder dos dados d20!

### ✨ Características Principais

- 🎭 **Criação de Personagem**: Crie seu herói único com nome personalizado
- ⚔️ **Sistema de Combate**: Batalhas épicas usando dados d20 para determinar o vencedor
- 🏪 **Sistema de Economia**: Colete moedas e compre poções mágicas
- 🌲 **Múltiplas Fases**: Explore floresta, vilarejo, clareira mística e arena final
- 🎯 **Escolhas Importantes**: Suas decisões afetam o rumo da aventura
- 📊 **Progressão**: Sistema de vida, força e recursos
